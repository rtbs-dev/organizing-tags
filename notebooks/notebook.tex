\subsubsection{Implied Component

Given our matrix A, where the rows are components and the columns are
contexts in which we might observe those components, we would like to
generate an adjacency matrix B that describes the co-occurrence of
components, by context, i.e., component \(x\) and \(y\) both are
connected to the parameter \(j\), so the parameter adjacency matrix
shows \(x\) and \(y\) are related. This is relatively simple to compute:

\(B = A\cdot A^{T}\)

\emph{NB: this is reverse of the traditional \(A^{T}\cdot A\), since our
column/row definitions are flipped.}
